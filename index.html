Executive Summary

What the audit found

The Google Ads audit for account 617-879-6307 (Painted Black) revealed a foundational tracking failure that has rendered all historical conversion data unreliable. The Google Ads base tag was never installed on the Wix website — meaning the account has been optimizing against phantom data since launch.

In addition, 14 of 16 configured conversion actions are broken, misconfigured, or belong to unrelated businesses sharing the account (Bergamot Media, Senstylable). Of two active campaigns, the Smart Campaign spent $128.87 with zero verified conversions, while the Performance Max Local campaign recorded 136 conversions at a $1.62 CPA — figures that cannot be trusted without a working tag.

This Plan of Action addresses every finding in three sequential phases: foundation & tracking repair, campaign overhaul, and ongoing optimization. Phases must be executed in order. Phase 2 work is only meaningful once Phase 1 is verified complete.

01
Phase 01 — Foundation

Tracking & Base Fix

Nothing in Phase 2 or 3 should be actioned until this phase is complete. Operating campaigns without a working tag means smart bidding trains on bad data and every optimization is made blind.

Task 1.1
Install the Google Ads base tag on Wix
  • Go to Google Ads → Tools → Conversions → Google tag — copy your Tag ID (AW-XXXXXXXXX)
  • In Wix: Settings → Custom Code → Add Code → paste in <head> on all pages
  • Or deploy via Google Tag Manager for cleaner long-term management
  • Verify using Chrome DevTools → Network tab → filter for "googleads" or "gtag"
  • Allow 24–48 hours for Google to confirm tag is active in account
Task 1.2
Rebuild conversion actions from scratch
  • Pause all 14 broken/foreign conversion actions (do not delete — preserve historical view)
  • Create 3 clean conversion actions for Painted Black: Directions click, Reservation intent (call/link/tap), Menu engagement (secondary)
  • Set counting method to "One" per conversion for reservation/lead goals
  • Attribution window: 7-day click, 1-day view — appropriate for restaurant discovery intent
Task 1.3
Link GA4 to Google Ads
  • Create a GA4 property for paintedblack.club if one does not exist
  • Install via Wix native integration (Settings → Integrations → Google Analytics) or GTM
  • In Google Ads: Tools → Linked Accounts → Google Analytics → link the GA4 property
  • Enable Google signals for audience features
  • After 30 days, GA4 audiences can power remarketing and bidding signal improvements
02
Phase 02 — Rebuild

Campaign Overhaul

Phase 2 begins only after the base tag is verified active and at least 2 of 3 new conversion actions are confirmed firing. Starting Phase 2 before Phase 1 means smart bidding continues training on corrupted data.

Task 2.1
Pause Smart Campaign → Launch Standard Search
  • Pause the Smart Campaign — $128.87 spent with zero verified conversions
  • Replace with a Standard Search campaign targeting reservation-intent keywords
  • Core keywords: "restaurant in tirana", "italian restaurant tirana", "tirana dinner", "best restaurants tirana"
  • Use exact match and phrase match initially — broad match too unpredictable for single-location
  • Set bidding to Maximize Clicks first — switch to Target CPA once 30+ conversions recorded
Task 2.2
Performance Max Local — retain with fixes
  • Keep paused until base tag is verified — then re-enable
  • Reassign campaign to optimize toward new clean conversion actions (Directions, Reservations)
  • Upload 3–5 high-quality restaurant photos as image assets (food, interior, ambience)
  • Add minimum 5 headline and 5 description variants to asset group
  • Review auto-targeting to exclude irrelevant categories
Task 2.3
Resolve alcohol policy violation
  • Review all ad copy for references to wine, cocktails, beer, spirits, or liquor
  • Remove or soften alcohol references in headlines, descriptions, and site link extensions
  • If the menu page is landing page and features drinks prominently, redirect to a food-focused page
  • Submit for review — typically clears within 1–3 business days once changes are made
Task 2.4
Landing page conversion audit
  • Ensure paintedblack.club has a clear primary CTA above the fold — reservation link or phone number
  • Test page load speed on mobile using Google PageSpeed Insights — target under 3 seconds
  • Walk through the full reservation flow on mobile to confirm it functions end-to-end
  • Add click tracking on the reservation button so it can fire a conversion event
Note
Phase 2 campaign work cannot begin until Phase 1 tag installation is confirmed verified in the Google Ads UI. Launching optimized campaigns against an untagged site restarts the data corruption problem from day one.
03
Phase 03 — Optimize

Scale & Refine

Phase 3 activates once clean conversion data has been collecting for at least 2–3 weeks. All optimization decisions must be based on verified post-fix data — not pre-fix historical figures from the broken account state.

Task 3.1
Switch bidding to Target CPA
  • Once Search campaign has 30+ conversions in a 30-day window, switch from Maximize Clicks to Target CPA
  • Set initial Target CPA at 150–200% of the current average CPA — then tighten by 10% every 2 weeks
  • Apply same strategy to PMax Local once conversion actions are verified clean
  • Never switch bidding strategies during the 1–2 week data accumulation period — it resets learning
Task 3.2
Search term hygiene & negative keywords
  • After 2 weeks of Search campaign data, download the Search Terms report
  • Add irrelevant or low-intent terms to a shared negative keyword list
  • Immediate negatives: job listings, recipes, cooking tutorials, competitor brand names
  • Review weekly for the first 6 weeks, then bi-weekly
Task 3.3
GA4 audience activation
  • Once GA4 has 30 days of data, build remarketing audiences: site visitors (last 30 days), menu viewers, engaged sessions > 90 seconds
  • Import audiences into Google Ads for RLSA bid adjustments on Search campaigns
  • Apply +30% bid adjustment for people who previously visited and are searching again
  • Build a "high-intent visitor" exclusion from Display campaigns to prevent wasted retargeting
